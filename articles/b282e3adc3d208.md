次の正規表現を、JSON内に記述したいとします。

```regex
^\[\d
```

`\` 自体を、 `\` でエスケープします。

```json
{
  "pattern": "^\\[\\d"
}
```

AWSでは、設定ファイルにJSONが使用されることが多いです。設定項目によっては、正規表現を記述できる場合があります。

JSONでも `\` は `"` などのエスケープに使用されます。JSONパーサーが `\\` を評価した結果 `\` になりますので、結果的に正しい正規表現のメタ文字が渡ります。
